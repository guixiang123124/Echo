Echo 隐私政策

生效日期：2026-02-06

1. 我们是谁

Echo（以下简称“我们”）提供语音输入与文本转换服务。

2. 我们处理哪些数据

  • 语音数据（用于语音识别与转写）
  • 转写文本（用于展示、插入与历史记录）

3. 数据如何使用

  • 完成语音识别与文本输出
  • 在你授权后插入到当前输入框
  • （可选）对转写文本进行自动润色

4. 数据存储与保留

  • 历史记录存储在本地设备（可配置保留策略）
  • 你可随时清理本地历史记录

5. 第三方服务

当你启用云端 ASR 或自动润色时，语音或文本会发送至第三方服务商进行处理（如 OpenAI 等）。这些服务商对数据的处理遵循其各自隐私政策。

6. 权限说明

  • 麦克风：用于采集语音
  • 键盘扩展：用于输入与插入转写文本

7. 数据安全

  • API Key 存储在系统安全存储（Keychain）
  • 我们采取合理措施保护数据安全

8. 你的权利

  • 关闭相关权限
  • 删除本地历史记录
  • 停用云端服务配置

9. 联系方式

如有问题，请联系：support@fashionflow.ai
