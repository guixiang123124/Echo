# App Privacy 问卷草案（用于 App Store Connect）

> 提示：这是草案，提交前请按你最终实现逐项确认。

## Data Types（建议初稿）

### Audio Data
- Collected: Yes
- Linked to user: No（若无账户体系）
- Used for tracking: No
- Purpose: App Functionality

### User Content（转写文本）
- Collected: Yes
- Linked to user: No（若无账户体系）
- Used for tracking: No
- Purpose: App Functionality

### Diagnostics（错误日志）
- Collected: Optional（若仅本地则可填 No）
- Linked to user: No
- Used for tracking: No
- Purpose: Diagnostics

## Third-Party Processing
- If using cloud ASR/LLM providers, declare external processing in privacy policy.

## Tracking
- Tracking: No（前提是没有广告追踪或跨 App 追踪）

## Account & Identity
- 若没有登录系统，通常可以不声明身份数据采集。
