## Description (Ready to paste)
Typeless is a macOS voice input assistant for fast text entry.

Place the cursor in any text field, then:
1. Start recording
2. Stop recording
3. Get transcribed text inserted automatically

Key features:
- Global recording trigger
- Transcription after each recording session
- Auto-insert into the active text field
- Local history and playback
- Local SQLite-based storage

Permissions used:
- Microphone: capture speech audio
- Input Monitoring: detect global trigger key
- Accessibility: insert transcribed text into the focused input field

Ideal for messaging, writing, support workflows, and meeting notes.
