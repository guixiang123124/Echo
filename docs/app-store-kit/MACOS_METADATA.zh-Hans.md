## 描述（可直接粘贴）
Echo 是一款面向 macOS 的语音输入助手。

在任意输入框中放置光标后：
1. 触发录音
2. 再次触发结束
3. 几秒后自动插入转写文本

核心能力：
- 全局触发录音
- 录音结束后自动转写
- 自动插入当前光标位置
- 历史记录与本地回放
- 本地 SQLite 存储

权限说明：
- Microphone：采集语音
- Input Monitoring：检测全局触发键
- Accessibility：将转写文本插入当前输入框

适用场景：聊天、文档写作、客服回复、会议纪要等。
