# 01 账号与法务清单

## A. 账号类型选择
- [ ] 个人上架：使用个人 Apple Developer 账号（显示个人名）
- [ ] 公司上架：使用 Organization 账号（显示公司名）

## B. 必做前置
- [ ] Apple ID 开启双重认证（2FA）
- [ ] 加入 Apple Developer Program（付费会员）
- [ ] 在 Xcode 中能看到 Team ID

## C. 若走公司账号（Organization）
- [ ] 公司是合法实体（Legal Entity）
- [ ] 有 D-U-N-S 编号
- [ ] 有公司域名邮箱（非个人邮箱）
- [ ] 有可访问的公司官网
- [ ] 法律授权联系人可接收验证电话/邮件

## D. App Store Connect 商务配置
- [ ] 签署最新协议（Agreements）
- [ ] 完成税务信息（Tax）
- [ ] 完成收款银行信息（Banking）

## E. 你必须亲自完成的动作
- [ ] 接受协议
- [ ] 提交税务表单
- [ ] 提交银行信息
- [ ] 最终点击“提交审核”

## F. 我可以继续代你做的动作
- [ ] 整理全部上架文案
- [ ] 校验 Info.plist / Entitlements / Capabilities
- [ ] 生成审核备注、测试账号说明、隐私问卷草稿
- [ ] 生成提审前自检清单并逐项验收
