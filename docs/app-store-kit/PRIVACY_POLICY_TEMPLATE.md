# 隐私政策模板（草案）

> 注意：此模板用于产品准备，不构成法律意见。上线前建议请律师或合规顾问复核。

## 1. 我们是谁
Echo（以下简称“我们”）提供语音输入与文本转换服务。

## 2. 我们收集哪些数据
根据你启用的功能，我们可能处理以下数据：
- 语音数据（用于语音转文字）
- 转写文本（用于展示、插入与历史记录）
- 基本诊断日志（用于故障排查）

## 3. 数据如何使用
- 完成语音识别与文本输出
- 在你授权后插入到当前输入框
- 改善识别质量与产品稳定性

## 4. 数据存储与保留
- 录音与历史记录可存储在本地设备（SQLite + 本地文件）
- 默认可配置保留策略（例如 7 天）
- 你可清理本地历史记录

## 5. 第三方服务
如果你配置第三方模型服务（如 OpenAI），音频或文本会发送至对应服务商进行处理，受其隐私政策约束。

## 6. 权限说明
- 麦克风：用于采集语音
- 输入监听（macOS）：用于检测全局热键
- 辅助功能（macOS）：用于把转写文本插入到当前输入框

## 7. 数据安全
- API Key 存储在系统安全存储（Keychain）
- 我们采取合理措施保护数据安全

## 8. 你的权利
你可随时：
- 关闭相关权限
- 删除本地历史记录
- 停止使用相关云服务配置

## 9. 儿童隐私
本产品不面向未满 13 岁儿童。

## 10. 政策更新
我们可能更新本政策，并在产品或官网公布最新版本。
